# 🚀 Cybersecurity Toolkit

A modular command-line cybersecurity toolkit built in Kali Linux.  
This tool combines multiple security utilities into one system for learning and practical use.

---

## 🔥 Features

### 🔎 Port Scanner
- Multithreaded TCP port scanning
- Detects open ports
- Basic service identification (HTTP, SSH, etc.)
- Saves scan results

### 🔐 Password Strength Checker
- Checks password strength
- Uses regex-based validation

### 📂 File Integrity Checker
- Uses SHA256 hashing
- Detects file tampering

---

## 🛠️ Tech Stack

- Python 3
- Socket Programming
- Multithreading (`ThreadPoolExecutor`)
- Hashing (`hashlib`)
- Regular Expressions
